TDesign组件库

发表于 2021-03-12 | 更新于: 2021-03-12 | 分类于 TDesign

TDesign 是公司级前端通用UI组件库，由前端通用 UI 组件库 Oteam 发起，协同公司内有组件库开发经验的同学一起建设的组件库，使用 TypeScript 开发，享受 TS 社区带来的各种便利，统一的设计语言及视觉风格，支持多端、多技术栈，包括设备终端（PC/移动）、运行平台（Web/小程序）、前端框架（React/Vue.js/Angular）

应用场景

TDesign 拥有一套完善的设计语言和设计价值观，在设计过程中广泛征求了公司多方意见，力求打造腾讯整体品牌形象。如果希望产品风格与公司更加统一，或者产品形态没有太多个性化要求，TDesign UI组件库提供的主题编辑器能够满足一般性的定制化需求。

组件库分类

TDesign Vue

特性

  • 前端通用 UI 组件库 Oteam Web 版实现
  • PC端基于 Vue2.0（3.0 升级规划中）
  • 移动端基于 Vue3，并使用 Vite 2.x 开发构建
  • 对接前端组件语言规范
  • 与其他框架（React）API、 UI 保持一致
  • 支持国际化（完善中）
  • 支持按需加载

  • 安装

组件

Menu 导航菜单

相对于之前使用的其他组件库，功能更多例如侧边菜单展开，收起，添加Logo等

Pagination 分页

分页组件常与表格组件一同使用，分页大小一般根据项目需求展示与隐藏，其中可以通过 pageSizeOption 属性控制，传 [] 表示不显示分页大小选择器，默认参数是 [10,20,50,100] ，还自定义数据如：[{ label: ‘每页 5 条’， value: 5 }, { label: ‘每页 10 条’, value: 10 } }] 参数

Steps 步骤条

带额外内容步骤条，适用于步骤中需要额外操作，等待下一步操作场景

Upload 上传

批量上传通常作为一个单独的功能出现，上传的内容用表格来承载。由于批量上传需要的时间较长，某些场景下会出现先选择文件，确认后再统一上传的场景。
